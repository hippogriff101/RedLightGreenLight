# RedLightGreenLight
Red Light Green Light with a ESP32!

[See demo video on Macondo system!](https://youtube.com)

This was submitted to Hack Club's Macondo

## What you need!
- ESP32-WROVER x1
- GPIO Extension Board x1
- LED x2 (Red & Green)
- Resistor 220Ω x2
- Jumper M/M x6
- Resistor 10kΩ x2
- Push button x1

- Computer with Python (1.12 or higher i think)
  
_you can proberbly do without all of this but this is just what I have!
